Project 3: Smart Motorcycle Helmet and App

Computer Engineering 186: Intro to Computer Engineering II

The third project was for a semester long project class that was done in a group of three. We were allowed to choose any topic for our project. This project was built with Flutter (a Google cross-platform app development framework). We used Arduino Nano sensors combined with a crash detection algorithm to broadcast an SMS text message. This was done using Bluetooth low energy amd it was sent to a list of emergency contacts using Twilio API in the event a motorcyle crash was detetced. Through this project I learned Dart and the framework Flutter. I also leanred a bit of C++ through the Arduino IDE. Lastly, I learned the basics of Git.
